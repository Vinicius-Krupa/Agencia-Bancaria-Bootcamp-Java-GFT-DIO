# 🎯Objetivo

Concluir o desafio de projeto do BootCamp Java da empresa GFT, criando um sistema de agência bancaria capaz de realizar as principais operações de um banco.

---

# 🔧Funcionalidades

- Criar Conta;
- Ver Saldo;
- Depositar;
- Sacar;
- Transferir;
- Listar Contas;
- Sair.

---

# ✔️Técnicas Tecnologias Utilizadas

- Java.
